TEnmo

PostgreSQL - RESTful API - Java - JDBC template - Spring Boot - OOP

A Venmo-clone application created to simulate cash-trading between users.

This was my second project at Tech Elevator. I was paired with 1 other developer and we were tasked with creating a Venmo clone which would allow authorized users to trade cash between other authorized users. We used a JDBC template and PostgreSQL to create communication between a server and database. HTML - CSS - Javascript - Vue.js - Axios - Java - PostgreSQL - RESTful API - Java - JDBC template - Spring Boot - IntelliJ IDEA - OOP - Agile

A social media platform for meal planning and recipe archiving/sharing. Express your own unique brand of ideas and connect with other cultures and experiences across the entire planet through archiving and sharing recipes on Worldbyte.

Created as a final capstone project for Tech Elevator Bootcamp.
